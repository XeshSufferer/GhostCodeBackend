# User Content Service

## Актуальные эндпоинты:

### GET /link/avatars/{avatarLink}

avatarLink - Код изображения аватара, приходит в UserData - объекте в поле avatarLink

Возвращает:
 - 200 - Скачивает картинку, можно вставлять в html и т.д

### GET /link/headers/{headerLink}

headerLink - Код изображения аватара, приходит в UserData - объекте в поле headerLink

Возвращает:
 - 200 - Скачивает картинку, можно вставлять в html и т.д


### POST /uploadAvatar

**ТРЕБУЕТ ВАЛИДНЫЙ JWT**

Принимает изображение которое будет установлено в качестве аватара.

Возвращает:
 - 400 - Error uploading avatar. Ошибка JWT (Оповещать разработчика бекенда, существование этого ответа - критический баг)

 - 200 - Успешная установка аватара, возвращает код изображения аватара. Ответ в формате JSON

 ```
    {
        "link": "string"
    }
 ``` 

### POST /uploadHeader

**ТРЕБУЕТ ВАЛИДНЫЙ JWT**

Принимает изображение которое будет установлено в качестве аватара.

Возвращает:

 - 400 - Error uploading header. Ошибка JWT (Оповещать разработчика бекенда, существование этого ответа - критический баг)

 - 200 - Успешная установка шапки профиля, возвращает код изображения шапки профиля. Ответ в формате JSON

 ```
    {
        "link": "string"
    }
 ``` 

===

## Rate Limits ⚡

- **Upload-операции**: 3 запроса в минуту
- **При превышении**: таймаут на 30 минут

### Важные замечания:
- Лимит применяется ТОЛЬКО к операциям загрузки (/uploadAvatar, /uploadHeader)
- Получение файлов (GET /link/...) не ограничено
- Будьте осторожны с автоматической загрузкой нескольких файлов
